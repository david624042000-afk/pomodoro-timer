# 🍅 番茄工作法計時器 (Pomodoro Timer)

一個簡潔好用的網頁版番茄工作法計時器，幫助你提高工作效率。

## 功能特色 ✨

- **時間預設** - 工作時間和休息時間有四個常用預設 (15/25/45/60分鐘)
- **自訂時間** - 可以設定任意時長的工作和休息時間
- **統計追蹤** - 記錄今日完成的工作數、連續天數和專注總時間
- **聲音提醒** - 計時器完成時會播放聲音通知
- **中英雙語** - 完整中文介面搭配英文說明

## 快速開始 🚀

1. 在瀏覽器中打開 `pomodoro/index.html`
2. 點擊「開始」按鈕開始工作計時
3. 時間結束後，系統會自動切換到休息時間
4. 點擊「⚙️ 設置」可自訂工作和休息時間

## 使用說明 📖

### 基本操作
- **開始 (Start)** - 開始計時
- **暫停 (Pause)** - 暫停計時  
- **重置 (Reset)** - 重置為預設時間

### 自訂設置
在設置面板中：
1. 選擇工作時間 (Work Duration)
   - 預設選項：15、25、45、60分鐘
   - 或輸入自訂分鐘數
2. 選擇休息時間 (Break Duration)  
   - 預設選項：5、10、15、30分鐘
   - 或輸入自訂分鐘數
3. 點擊「保存」套用設置

### 統計信息
- **完成工作數** - 今日完成的番茄鐘數量
- **今日連續 🔥** - 連續完成的工作數
- **今日專注時間** - 累積的專注工作時間
